# Connector Contract

Every connector must support:

- mock mode
- read mode
- draft mode
- execute mode
- health checks
- rate limiting
- permission checks
- audit logs
- dry-run mode
- error handling

Current placeholder connectors:

- OpenAI
- Supabase
- Stripe
- Twilio
- Gmail
- Google Calendar
- GitHub
- Vercel
- Cloudflare

Execution methods remain blocked until credentials, scoped approval, and verification are provided.
